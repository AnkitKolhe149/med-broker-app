# Medicine Marketplace Platform

This is a comprehensive platform for medicine marketplace, including frontend, backend, AI-ML, and infrastructure components.

## Structure

- `frontend/`: React (Vite) application
- `backend/`: Node.js backend
- `ai-ml/`: Python AI/ML services
- `infrastructure/`: Docker, Nginx, and cloud configurations
- `docs/`: Documentation

## Getting Started

1. Clone the repository
2. Navigate to each directory and follow setup instructions
